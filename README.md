<h1>🚀 Interactive Task Dashboard</h1>
  </p> A modern React-based task management dashboard demonstrating state management,
    dynamic rendering, and clean component architecture.
  </p>
  <h2>📌 Project Description</h2>
  <p>
    The <strong>Interactive Task Dashboard</strong> is built using React functional components.
    It allows users to create, delete, and manage tasks dynamically.
    The project demonstrates the use of React Hooks such as <strong>useState</strong> and <strong>useEffect</strong>.
  </p>

  <h2>✨ Features</h2>
  <ul>
    <li>✅ Add tasks dynamically</li>
    <li>❌ Delete tasks</li>
    <li>🔄 Toggle task completion</li>
    <li>🌙 Dark / ☀️ Light theme toggle</li>
    <li>📊 Task count summary</li>
    <li>🎨 Stylish modern UI</li>
  </ul>

  <h2>🛠 Technologies Used</h2>
  <ul>
    <li>React (Functional Components)</li>
    <li>useState & useEffect Hooks</li>
    <li>CSS3 (Modern Styling)</li>
    <li>Vite</li>
  </ul>

  <h2>⚙️ Installation & Setup</h2>
  <ol>
    <li>Clone the repository</li>
    <li>Open the project folder</li>
    <li>Run <code>npm install</code></li>
    <li>Run <code>npm run dev</code></li>
    <li>Open browser at <code>http://localhost:5173</code></li>
  </ol>

  <h2>🎯 Learning Outcomes</h2>
  <ul>
    <li>Understanding React component architecture</li>
    <li>Managing state using useState</li>
    <li>Handling side effects using useEffect</li>
    <li>Implementing conditional rendering</li>
    <li>Creating responsive UI design</li>
  </ul>
