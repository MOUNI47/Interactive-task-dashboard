  <p align="center">

  </p>

  <hr>

  <h2>📌 Project Description</h2>
  <p>
    The <strong>Interactive Task Dashboard</strong> is built using React functional components.
    It allows users to create, delete, and manage tasks dynamically.
    The project demonstrates the use of React Hooks such as <strong>useState</strong> and <strong>useEffect</strong>.
  </p>

  <h2>✨ Features</h2>
  <ul>
    <li>✅ Add tasks dynamically</li>
    <li>❌ Delete tasks</li>
    <li>🔄 Toggle task completion</li>
    <li>🌙 Dark / ☀️ Light theme toggle</li>
    <li>📊 Task count summary</li>
    <li>🎨 Stylish modern UI</li>
  </ul>

  <h2>🛠 Technologies Used</h2>
  <ul>
    <li>React (Functional Components)</li>
    <li>useState & useEffect Hooks</li>
    <li>CSS3 (Modern Styling)</li>
    <li>Vite</li>
  </ul>
